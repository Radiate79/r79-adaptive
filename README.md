# R79 Adaptive

A small adaptive setup lab for motorsport wheel, car, and track combinations.

## Development

1. Install dependencies with `npm install`.
2. Start the app with `npm run dev`.
3. Run `npm run lint` and `npm run build` before shipping.

## Deployment

This project is configured for GitHub Pages.

1. Ensure the repo name matches the Vite base path in `vite.config.js`.
2. Run `npm run deploy` to build the app and publish the `dist/` folder to the `gh-pages` branch.
